Задание 1

Напишите функцию calculate_circle_area(radius), которая принимает радиус круга и возвращает его площадь

S=π*radius^2

Задание 1

Ответ


def calculate_circle_area(radius):
    if radius < 0:
        return "Ошибка: радиус не может быть отрицательным."
    pi = 3.14
    return pi * radius ** 2

area = calculate_circle_area(5)

print("Площадь круга:", area)  # Площадь круга: 78.5
print(calculate_circle_area(-2))  # Ошибка: радиус не может быть отрицательным.

Задание 2

Напишите функцию is_even(number), которая принимает целое число и возвращает True, если число четное, и False в противном случае.

Операторы сравнения возвращают логический тип значений

Задание 2

Ответ


def is_even(number):
    return number % 2 == 0

print(is_even(4))  # True
print(is_even(7))  # False

def filter_even_numbers(numbers):
    return [num for num in numbers if is_even(num)]

print(filter_even_numbers([1, 2, 3, 4, 5, 6]))  # [2, 4, 6]

Задание 3

Создайте функцию celsius_to_fahrenheit(celsius), которая принимает температуру в градусах Цельсия и возвращает её в градусах Фаренгейта

F = C * (9/5) + 32. C = (F - 32) * 5/9

Задание 3

Решение


def celsius_to_fahrenheit(celsius):
    return celsius * 9 / 5 + 32

def fahrenheit_to_celsius(fahrenheit):
    return (fahrenheit - 32) * 5 / 9

# Примеры вызова
print(celsius_to_fahrenheit(25))  # 77.0
print(fahrenheit_to_celsius(77))  # 25.0

Параметры по умолчанию

Параметры по умолчанию задают значение аргумента, которое используется, если вызов функции не передал его явно 


def greet(name, greeting="Привет"):
    print(f"{greeting}, {name}!")

greet("Анна")              # Привет, Анна!
greet("Анна", "Здравствуйте")  # Здравствуйте, Анна!

Задание 4

Напишите функцию convert_currency(amount, rate=70), которая принимает сумму в долларах и курс валюты (по умолчанию 70), а затем возвращает сумму в рублях

Задание 4

Решение


def convert_currency(amount, rate=70):
    return amount * rate

# Примеры вызова
print(convert_currency(100))       # 7000
print(convert_currency(100, 75))  # 7500

Задание 5

Напишите функцию calculate_discount(price, discount=10), которая принимает цену товара и процент скидки (по умолчанию 10%), а затем возвращает финальную цену после скидки.

Задание 5


def calculate_discount(price, discount=10):
    return price - (price * discount / 100)

# Примеры вызова
print(calculate_discount(1000))       # 900.0
print(calculate_discount(1000, 20))  # 800.0

Задание 6

Создайте глобальную переменную counter, которая увеличивается на 1 при каждом вызове функции increment_counter()

Задание 6


# Глобальная переменная
counter = 0

def increment_counter():
    global counter  # Указываем, что используем глобальную переменную
    counter += 1
    print(f"Counter: {counter}")

# Примеры вызова
increment_counter()  # Counter: 1
increment_counter()  # Counter: 2
increment_counter()  # Counter: 3
